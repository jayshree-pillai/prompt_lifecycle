# Company Overview (v2025_01_10)

Write the **Company Overview** section for the issuer.

## Inputs available
- SEC-style filings (10-K / 10-Q) or equivalent financial disclosures
- Internal ratings files (if provided)
- Any provided JSON/company profile fields (if provided)

## What to cover (in this order)
1. **Business description**
   - What the company does, primary products/services, and how it makes money.
2. **Segments / geography**
   - Main operating segments and key geographic exposures (if disclosed).
3. **Competitive position**
   - Market position, scale, differentiation, key competitors (only if supported).
4. **Recent developments**
   - Major events in the last 12–18 months: acquisitions, divestitures, restructurings, notable strategy shifts (only if supported).

## Must-follow rules
- If a detail is missing, write: **"Not found in provided materials."**
- Do not guess market share, rankings, or competitor names unless explicitly present.
- Keep it to ~8–12 bullets total.

## Output format
**Company Overview**
- Bullet 1 … **Evidence:** <short snippet or source label>
- Bullet 2 … **Evidence:** <short snippet or source label>
